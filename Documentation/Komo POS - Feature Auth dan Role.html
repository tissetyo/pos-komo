User Persona

Kasir

Orang yang ada di depan counter dan handle transaksi customer langsung. Butuh akses cepat ke POS tanpa ribet — login sekali, langsung bisa kerja. Gak perlu akses ke back-office atau analytics.

Manager

Ngurusin operasional harian. Bisa akses back-office untuk kelola produk, menu, dan tim. Tapi gak bisa ubah billing atau setting akun utama.

Owner

Akses penuh ke semua fitur termasuk konfigurasi system, feature toggle, dan laporan keuangan.

Dapur

Hanya bisa lihat antrian order yang masuk. Gak perlu akses ke kasir atau back-office.

Detail Fitur

Login dan Session

Halaman login adalah screen pertama yang muncul saat POS dibuka. User masukkan email dan PIN (bukan password panjang — kasir butuh input cepat). Setelah berhasil, sistem validasi role lalu redirect ke dashboard sesuai role. Session bertahan sampai kasir logout manual atau session expired.

Edge cases yang perlu dihandle:

  • PIN salah 5x — lock akun sementara, notifikasi ke Manager
  • Session expired saat lagi di tengah order — simpan draft order, minta login ulang
  • User yang di-deactivate — langsung logout saat next request

Role dan Permission

Role bawaan yang tersedia:

  • Owner — akses penuh, termasuk billing, feature toggle, dan hapus outlet
  • Manager — kelola produk, menu, outlet, user, dan lihat analytics
  • Kasir — akses ke layar kasir saja, tidak bisa masuk back-office
  • Dapur — hanya lihat antrian order, tidak bisa bikin atau edit order

Manager bisa bikin custom role dengan kombinasi permission yang berbeda. Role di-assign per user per outlet — satu user bisa jadi Manager di Outlet A tapi Kasir di Outlet B. Perubahan role berlaku saat session berikutnya (login ulang).
