Detail Fitur

Modul yang Bisa Di-toggle

Di kasir:

  • Real-time order status (WebSocket ke dapur)
  • Print struk otomatis
  • Cash drawer integration

Di back-office:

  • Analytics Dashboard
  • Multi-outlet management
  • Printer settings

Cara Kerja

Owner mengakses halaman Feature Toggle di Settings akun. Setiap modul ditampilkan dengan nama, deskripsi singkat, dan toggle switch. Kalau Owner hide suatu modul, modul tersebut tidak tampil di navigasi kasir atau back-office untuk semua user di semua outlet. Kalau show kembali, langsung muncul saat page di-reload. Fitur ini hanya bisa diakses Owner — Manager tidak bisa mengubah feature toggle.

Hal yang Perlu Dikonfirmasi

  • Apakah feature toggle bisa diset berbeda per outlet?
  • Apakah ada notifikasi ke user lain kalau ada fitur yang baru di-enable atau di-disable?
