Phase 1 — Core POS (Prioritas Sekarang)

Fitur dasar kasir dan back-office yang harus ada sebelum bisa dipakai.

  • Login dan session management per role
  • Tampilan menu kasir — card touch-friendly
  • Buat dan kelola open order (multi-order)
  • Pilih varian dan modifier produk
  • Pembayaran QRIS dan EDC
  • Print struk otomatis
  • CRUD produk di back-office
  • Varian dan modifier produk
  • Role management (Owner, Manager, Kasir, Dapur)

Jira Epic: Auth & Role, Kasir POS, Produk

Total SP Phase 1: 41

Phase 2 — Offline Mode dan Realtime

Operasional tetap jalan tanpa internet, update real-time antara kasir dan dapur.

  • Offline order dengan IndexedDB
  • Auto sync dan manual sync saat reconnect
  • WebSocket real-time order status

Jira Epic: Offline Mode

Total SP Phase 2: 21

Phase 3 — Multi-outlet dan Analytics

Kelola banyak toko, pantau performa bisnis.

  • Setup multi-outlet dari satu akun
  • Assign produk per outlet
  • Toggle visibilitas produk per outlet
  • Dashboard revenue (total, top produk, tren harian)

Jira Epic: Outlet, Analytics, Produk (toggle)

Total SP Phase 3: 22

Phase 4 — Onboarding dan Feature Toggle

First impression yang baik untuk user baru, dan fleksibilitas konfigurasi tampilan.

  • Setup wizard step-by-step
  • Setup printer di onboarding
  • Show/hide fitur per tipe resto

Jira Epic: Onboarding, Feature Toggle

Total SP Phase 4: 16

Phase 5 — Integrasi dan Design System (Later)

Integrasi pihak ketiga, design system final, migrasi Komo bertahap.

  • Third-party integrations (delivery, loyalty, dll)
  • Design system finalisasi
  • Migrasi data Komo existing
  • Cash drawer — perlu konfirmasi kebutuhan dulu

Total SP Phase 5: TBD
